# HMPI Calculator

## Overview
A PHP-based web application that calculates HMPI values based on user health parameters and generates reports.

## Features
- User-friendly interface
- Health parameter input
- HMPI calculation
- Report generation
- Location map integration

## Technologies Used
- PHP
- HTML
- CSS
- JavaScript

## Project Structure
home.php
input.php
logi.php
map.php
report.php
